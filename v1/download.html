Intel® RealSense™ Experience for Web Browsers (Windows 64bit)

 

​You must be at least 18 years old to install the plug-in on your computer.




  1. Make sure you have an Intel® RealSense™ system or an Intel® RealSense™ peripheral camera connected to a USB 3 port on your computer.
  2. Make sure you have the latest version of the camera driver.
  3. If you are at least 18 years old, check the checkbox to enable the download button, if not, get an adult to help you install the plug-in.
  4. Click on the Download button to save the file to your computer.
  5. Click on the downloaded file to start the installer and follow the instructions on the screen.
  6. After the installation is complete, close the installer and click here to continue straight to ScratchX or here to go back to the extension page.

System Requirements

Required Hardware

4th generation (or later) Intel® Core™ processor

2GB free hard disk space

Supported systems: Ultrabook™ devices, notebooks, 2 in 1s, and all-in-one PCs

An Intel® RealSense™ camera F200 (system-integrated or peripheral version)

 

Required OS

Microsoft Windows 8.1 OS 64-bit Desktop Mode

 

Supported Intel® RealSense™ Cameras

F200 – Front-facing

 

Supported Browsers

Microsoft Internet Explorer 10.0.13

Google Chrome 33.0.1750.146

Mozilla Firefox 27.0.1
