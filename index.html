Welcome Scratchers

Intel® RealSense™ Extension for Scratch introduces these amazing capabilities: hands tracking, gesture recognition, face tracking, face expressions and voice commands - all made simple with just a few Scratch blocks.

Scratch is a project of the Lifelong Kindergarten group of the MIT Media Lab

Getting Started

 To make the Scratch cat follow your index finger around:

  1. Choose Cat1 from the Sprite library
  2. Drag the Green Flag event and connect a “forever” loop
  3. From Motion, drag “set x to” and “set y to” blocks into the loop
  4. Go to More Blocks and drag a hand position block into each “set” block
  5. Set the position axis of each hand tracking block to the corresponding axis of the set block
  6. Make sure both tracking blocks are following the same hand and joint
  7. Press the green flag
  8. Present the appropriate hand to the camera, palm facing forward, fingers facing upwards, and move your hand around, the cat should be following your hand
