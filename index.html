Getting Started With Scratch ›

Tracking the index finger

  1. Choose Cat1 from the Sprite library
  2. Drag the green flag block and connect a forever block under it
  3. From Motion, drag set x to and set y to blocks under the forever block
  4. Go to More Blocks and drag a hand position block into each set block
  5. Set the position axis of each hand hand position block to the corresponding axis of the set block
  6. Make sure both hand position blocks are following the same hand and same joint
  7. Press the green flag
  8. Present your hand to the camera, palm facing forward, fingers facing upwards, and move your hand around, the cat should be following your hand

 

Rotating with the head

  1. Choose Cat1 from the Sprite library
  2. Drag the green flag block and connect a forever block under it
  3. From Motion, drag Point in direction block under the forever block
  4. Go to More Blocks and drag the rotation of head block
  5. Set the rotation axis to Roll
  6. Press the green flag
  7. Tilt your head left and right in front of the camera to see the cat rotate accordingly

 

Change the size with a V Sign gesture

  1. Choose Cat1 from the Sprite library
  2. Drag the green flag block and connect a forever block under it
  3. From Control, drag an if/else block under the forever block
  4. Go to More Blocks and drag the gesture block as the condition of the if
  5. Go to looks and drag a set size to blocks under the if and the else
  6. Set the set size block under if to 200%, keep the other one at 100%
  7. Press the green flag
  8. Present your hand to the camera and make a "V sign" with your index and middle fingers. the cat should change size when you perform the gesture, and get back to it's original size when you open your palm

User Experience (UX)

  1. Choose the Beetle from the Sprite library
  2. Drag the green flag block and connect a forever block under it
  3. From Control, drag an if block under the forever block
  4. Go to More Blocks and drag the User Said block as the condition of the if
  5. Write the word left in the string box
  6. Go to motion and drag a turn left 15 degrees block under the if
  7. Set the degrees to 90
  8. Press the green flag
  9. Say Left and watch the Beetle turn left.
